How to Export These Posters as Images

1. Open this file in Google Chrome (best rendering for fonts).
2. Set zoom to 100% (Ctrl+0).
3. For each poster: right-click → Inspect → find the .poster div → right-click → "Capture node screenshot" → saves as 1080×1080 PNG.
4. Or use Chrome DevTools → Ctrl+Shift+P → "Screenshot" → "Capture full size screenshot" for all posters on one sheet.

For print-quality exports: Use html2canvas or Puppeteer to render at 2× scale (2160×2160) for retina display.
